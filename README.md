# Task 1 – Data Cleaning

This repository contains the raw and cleaned datasets for Task 1.  
The data was cleaned using **Google Sheets**.

## Files Included
- **raw_data.csv** – Original dataset  
- **clean_data.csv** – Final cleaned dataset  

## Cleaning Steps (Summary)
1. Formatted names using `=PROPER()`
2. Standardized gender values (Male/Female)
3. Converted all dates to a consistent format (YYYY-MM-DD)
4. Removed duplicates (Data → Data cleanup)
5. Cleaned numeric columns (age, sales)

## Tools Used
- Google Sheets  
- Basic cleaning formulas  

The final cleaned dataset is ready for analysis.
